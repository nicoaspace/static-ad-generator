## Phase 1: Create the Brand DNA
Read the skill file at skills/SKILL.md. then run Phase 1 for LMNT. The brand URL is https://drinklmnt.com/ and the specific product is the LMNT Recharge Electrolyte Drink. Build the Brand DNA document and save it to brands/lmnt/brand-dna.md.

# Phase 2: Run Prompt Generation
Now run Phase 2. Read the template prompts file at skills/references/template-prompts.md and the Brand DNA at brands/lmnt/brand-dna.md. Fill in all 40 templates with LMNT-specific details for the Recharge Electrolyte Drink. Save the output as brands/lmnt/prompts.json.


# Phase 3: Run Image Generation
Now run Phase 3, but only for templates 1, 7, 9, 13, and 15. Read the skll file at skills/SKILL.md, then copy the generate_add.py script from skills/references/generate_ads.py into brands/lmnt/. Run the script with --templates 1,7,13,9,15 against brands/lmnt/prompts.json using the product images in brands/lmnt/product-images/.

  How it auto-locates everything:
  - PROJECT_ROOT = 3 levels above the script (static-ad-generator/)
  - brands/{brand}/prompts.json → prompts
  - brands/{brand}/product-images/ → reference images
  - brands/{brand}/outputs/ → where images are saved
  - env/.env.local → API key (no need to export every session)

  If --brand doesn't match, it lists all brands that have a prompts.json ready. No more copying the script into each brand folder.
